# 3. Upload the iOS building to TestFlight

*After open build folder and select latest build.
And click "Distribute App" button

![image](https://img001.prntscr.com/file/img001/PEgNHXJHScmn292gUJmt_Q.png)

*Select "TestFlight & App Store" option and click "Distribute" button

![image](https://img001.prntscr.com/file/img001/uOvHqUGkQ4C02eu2uhRiZQ.png)

# 4. Test the iOS building on TestFlight

*Move to the TestFlight and you can see new iOS building in TestFlight several minutes later from uploaded new building.

*You will see "Missing Compilance" warning message then click "Manage" button and select "None of the algorithms mentioned above" option and save.

![image](https://img001.prntscr.com/file/img001/33zfv7V3S327EaEdAEfCLg.png)
